# What Are the Exact Tasks We Complete — AI-Powered Sales

Each task is categorised by who owns it:
- **We** — completed by Valve One
- **Customer** — completed by the customer
- **Together** — requires active input from both sides

---

## Project Setup & Onboarding

| Task | Owner |
|------|-------|
| Grant HubSpot access to the delivery team | Customer |
| Introduce the key team members (sales lead, CRM admin, marketing owner) | Customer |
| Facilitate kickoff meeting and set delivery timeline | We |
| Run ICP workshop — who we're selling to, what triggers a purchase, common objections | Together |
| Define and validate the selling profile (offering, value props, tone of voice) | Together |
| Run data quality assessment on existing HubSpot data | We |
| Agree on lifecycle stage definitions (MQL, SQL thresholds) | Together |
| Agree on success metrics for the engagement | Together |
