# What Are the Exact Tasks We Complete — AI-Powered Service

Each task is categorised by who owns it:
- **We** — completed by us
- **Customer** — completed by the customer
- **Together** — requires active input from both sides

## Project Setup & Onboarding

| Task | Owner |
|------|-------|
| Grant HubSpot Service Hub access to the delivery team | Customer |
| Connect email inboxes, chat widget, and customer portal domain | Customer |
| Introduce key team members (service lead, CS lead, CRM admin) | Customer |
| Facilitate kickoff meeting and set delivery timeline | We |
| Run ticket taxonomy workshop — define categories, priorities, and routing logic | Together |
| Catalogue existing documentation and knowledge assets that can seed the knowledge base | Together |
| Run data quality assessment on existing HubSpot contact and company database | We |
| Agree on SLA targets: time to first response and time to close per ticket type | Together |
| Agree on success metrics for the engagement | Together |
